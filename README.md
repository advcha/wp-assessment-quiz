# Assessment Quiz

A WordPress plugin for creating and managing anxiety and depression assessment quizzes.

== Description ==

This plugin allows you to create, manage, and display assessment quizzes on your WordPress site. It includes features for managing quizzes, sections, questions, answers, and results.

== Installation ==

1. Upload the `wp-assessment-quiz` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the shortcode `[assessment_quiz id="your_quiz_id"]` to display a quiz on any post or page.
